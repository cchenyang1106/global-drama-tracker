# 2026-04-10 开发日志

## 搭建前端界面

为 Global Drama Tracker 项目搭建了完整的前端界面：

### 技术栈
- Vue 3 + Vite（端口 3000）
- Vue Router 4（路由管理）
- Pinia（状态管理）
- Element Plus（UI 组件库）
- Axios（HTTP 请求，配置了 `/api` 代理到后端 `localhost:8080`）

### 页面结构
- **首页** (`/`): Hero Banner + 今日更新 + 热门排行（日/周/月榜）+ 各地区热门
- **剧集库** (`/dramas`): 多条件筛选（地区、类型、状态、排序）+ 分页
- **剧集详情** (`/drama/:id`): 海报、评分、元数据、简介、外部链接
- **排行榜** (`/ranking`): 日榜/周榜/月榜/新剧榜，支持按地区筛选
- **搜索结果** (`/search?q=xxx`): 关键词搜索 + 分页

### 设计风格
- 暗色主题（深蓝色调）
- 现代卡片式布局
- 毛玻璃效果导航栏
- 平滑过渡动画
- 地区标签彩色编码

### 文件位置
- 前端目录: `frontend/`
- API 封装: `frontend/src/api/`
- 页面视图: `frontend/src/views/`
- 公共组件: `frontend/src/components/`（DramaCard, SectionHeader, RankingList）
- 全局样式: `frontend/src/styles/global.css`
- 常量定义: `frontend/src/utils/constants.js`

## Bug 修复
- **SearchResult.vue:19**: 修复模板字面量解析错误（模板字符串中的双引号与属性绑定冲突），改为字符串拼接
- **DramaServiceImpl.java**: 修复 `listDramas()` 中 `queryDto.getYear()` 为 null 时的 NPE。MyBatis-Plus 的 `.ge(condition, column, value)` 会先求值 `value`（即 `LocalDate.of(null, 1, 1)`）再判断 `condition`，改为 if 分支判断
- **Ranking.java**: `rank` 字段是 MySQL 保留字，使用 `@TableField("`rank`")` 包裹反引号，修复 SQL 语法错误
- **SecurityConfig.java**: 白名单添加 `/error` 端点。当 Controller 抛异常时 Spring 内部 forward 到 `/error`，缺少该端点会导致返回 403
- **Ranking.java + RankingServiceImpl.java**: 在 Ranking 实体中添加 `@TableField(exist = false) dramaTitle` 非持久化字段，查询排行榜后填充剧名供前端展示

## 数据修复与管理端分离

### 问题：首页/剧集库/排行榜无数据
- **今日更新**：`DramaServiceImpl.getTodayUpdated()` 使用 `DATE(update_time) = CURDATE()` 查询，种子数据的 update_time 不是今天导致返回空。**修复**：添加 fallback 逻辑，当 today 查询为空时返回 top 12 热门剧集（按 hotScore 降序）
- **排行榜**：Ranking 表为空（定时任务 cron 2:00am 未触发且用户未手动刷新）。**修复**：在 `DataController.initSeedData()` 接口中，种子数据初始化后自动调用 `rankingService.refreshRankings()`

### 管理端与用户端分离
- **新建** `frontend/src/layouts/AdminLayout.vue`：独立管理端布局，左侧边栏 + 顶栏 + 内容区，与用户端完全分离
- **重构** `frontend/src/router/index.js`：管理端路由 `/admin` 使用嵌套路由 + AdminLayout 组件，用户端路由保持不变
- **修改** `frontend/src/App.vue`：通过 `route.matched.meta.layout === 'admin'` 判断，管理端路由不渲染用户端导航栏/footer；用户端导航栏移除"数据管理"入口
- 管理端地址：`/admin`，管理端布局有"返回用户端"链接

## 环境说明
- 项目需要 **Java 17**（Spring Boot 3.2.4），本地 JDK 17 路径: `/opt/homebrew/opt/openjdk@17/libexec/openjdk.jdk/Contents/Home`
- 编译命令需设置 `JAVA_HOME` 为上述路径
- 前端构建: `cd frontend && npx vite build` ✓
- 后端编译: `JAVA_HOME=... mvn clean compile` ✓
